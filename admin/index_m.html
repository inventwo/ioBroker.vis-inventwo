inventwo Desing Widgets für die Visualisierung

Diese Seite dient nicht zur Konfiguration, sondern rein als Hilfeseite bzw. als eine kurze Anleitung!

Beim installieren einer Adapter-Instanz werden drei Datenpunkte angelegt.

Diese Datenpunkte können im VIS Editor für die Farbangaben genutzt werden.

Im Fall von der Instanz '0', kann der gewünschte Datenpunkt inkl. der geschweiften Klammer eingeben werden. (Solltest Du eine andere Instanznummer gewählt haben, muss die 0 natürlich gegen Deine Nummer ausgetauscht werden.)

Beispiel für die Farben:
{vis-inventwo.0.CSS.Button}
{vis-inventwo.0.CSS.Active}
{vis-inventwo.0.CSS.Text}

Für die gestreifeten Flächen, kann folgender Code in das Farbfeld eingetragen werden:
repeating-linear-gradient(-45deg, #333333 0px, #333333 15px, #393939 15px, #393939 30px, #333333 30px)
